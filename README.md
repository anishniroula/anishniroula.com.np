# anishniroula.com.np

Source code for my personal portfolio.

View it live at: [anishniroula.com.np](https://anishniroula.com.np)

## Overview

A minimalist, high-performance personal website. This project serves as a central hub for my professional portfolio, built entirely with vanilla web technologies to ensure zero dependencies and near-instant load times.

**Tech Stack:**

- **Languages:** HTML5, CSS3, JavaScript (ES6+)
- **Infrastructure:** Cloudflare Pages with GitHub Actions

## Local Setup

1. Clone the repo:
   ```bash
   git clone https://github.com/anishniroula/anishniroula.com.np.git
   ```
2. Open `index.html` in any browser.

## Structure

```text
├── index.html     # Main landing page
├── css/           # Custom stylesheets
├── js/            # Vanilla JavaScript logic
└── assets/        # Static images and design resources
```
